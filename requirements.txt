# Consulte a documentação do pdf2image ou pesquise como instalar o Poppler para o seu SO.
# Exemplos (dependendo do SO):
# No Linux (Debian/Ubuntu): sudo apt-get install poppler-utils
# No macOS (Homebrew): brew install poppler
# No Windows: Baixe os binários pré-compilados do Poppler e adicione-os ao PATH do sistema.
